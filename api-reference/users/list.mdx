---
title: 'List users'
api: 'Get /v1/users'
---

## Query parameters

<ParamField query="email" type="string" >
  Only include users whose email attribute equals this value.
</ParamField>
<ParamField query="segmentId" type="string" >
  Only include users in this segment. Find a segment's ID in the Usertour UI via the three-dot menu at the top right of the segment page.
</ParamField>
<ParamField query="companyId" type="string" >
 Only include users who are members of this company.
</ParamField>
