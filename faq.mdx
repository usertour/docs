## Technical Requirements

### Internet Connection

> **Note**: Usertour requires an active internet connection to function.

Usertour is a cloud-based solution that needs internet connectivity to:
- Load and display user experiences
- Track user interactions
- Update content in real-time
- Sync user data

### Deployment Options

#### Cloud Hosting (Recommended)
- ✅ Fully managed cloud service
- ✅ Automatic updates
- ✅ Built-in analytics
- ✅ Zero maintenance
- ✅ No AGPL compliance required
- ✅ Enterprise-grade support

#### Self-Hosted
- ⚠️ Requires AGPL v3 compliance
- ⚠️ **Enterprise Use**: Commercial license required for business/organizational use
- ✅ Complete data control
- ✅ Custom deployment options
- ✅ Private network support
- ✅ Custom domain support

### Application Support

#### Web Applications
- ✅ Cloud-based web applications
- ✅ Single-page applications (SPAs)
- ✅ Progressive web applications (PWAs)
- ✅ Mobile web browsers

#### Desktop Applications
- ❌ Local desktop applications (e.g., Outlook)
- ❌ Offline applications
- ❌ Third-party applications

#### Mobile Applications
- ❌ Native mobile apps
- ✅ Mobile web browsers
- ✅ Responsive web applications

### Integration Requirements

#### JavaScript SDK
- ✅ Easy to install via npm or CDN
- ✅ Works with any web application
- ✅ Supports modern web frameworks
- ✅ Lightweight and performant

#### Single-Page Applications
> **Note**: Usertour fully supports SPAs and automatically handles route changes. No manual configuration is required.

#### Product Experience Platform
> **Note**: Usertour is a product experience platform that helps you:
> - Guide users through your product with interactive tours
> - Create in-app checklists for task completion
> - Build feature launchers for easy access
> - Deliver contextual help within your application
