This page applies to self-hosted versions `0.5.0` and earlier.

If you're using a newer self-hosted version, use [License Management](/open-source/license-management) instead.

This page is kept only as a legacy reference for older self-hosted project subscription screens.

Usertour offers 3 self-hosted pricing plans: **Free, Business, Enterprise**. Both **Business** and **Enterprise** plans require purchasing a license. You can [contact us](https://tally.so/r/n904o1).


## Legacy Project License Upload

Navigate to the **Settings > Subscription**, click **Upload License** and upload your license.

<Note>
If you're using a version earlier than 0.3.0, you won't see this configuration option. Version 0.3.0 shows "Update License", while versions after 0.3.0 show "Upload License".
</Note>

![](/images/license-05.png)

After successful configuration, you'll see the **Current plan** change to **Business** or **Enterprise**, along with an expiration date displayed as **"Expires on [date]"**.
![](/images/license-06.png)
