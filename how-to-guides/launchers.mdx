## Overview

Launchers are powerful UI elements that help you:
- Highlight important features
- Provide quick access to help
- Guide users to key areas
- Create interactive tooltips
- Add visual cues to your interface

All without writing a single line of code.

## Creating Launchers

### Basic Setup
1. Sign in to [Usertour](https://app.usertour.io)
2. Navigate to **Launchers** in the sidebar
3. Click **Create launcher**
4. Name your launcher
5. Click **Create launcher** to proceed

Launchers are published automatically, just like flows and checklists.

## Customization

### Appearance
Customize your launchers through the theme editor:
- Access via [Settings -> Themes](https://app.usertour.io/project/1/settings/themes)
- Modify beacon styles
- Customize icon designs
- Adjust button appearances
- Match your app's design

### Layer Control
Control how launchers appear in your interface:

#### Z-Index Management
- Adjust launcher visibility
- Control layering with other elements
- Set display priority
- Configure in the Launcher Builder

## Best Practices

1. **Strategic Placement**
   - Place near important features
   - Ensure good visibility
   - Avoid cluttering the interface
   - Consider user workflow

2. **Visual Design**
   - Use consistent styling
   - Match your app's theme
   - Make them noticeable but not distracting
   - Ensure good contrast

3. **User Experience**
   - Keep tooltips concise
   - Provide clear actions
   - Make interactions intuitive
   - Test on different screen sizes

4. **Performance**
   - Use appropriate z-index values
   - Optimize for different devices
   - Consider loading times
   - Test in various scenarios

> **Tip**: Use launchers strategically to guide users to new or important features without overwhelming them.
